## 📖 О проекте

Lens & Look — это полностью адаптивный интернет-магазин оптики, разработанный в рамках учебного проекта. Сайт предоставляет удобный каталог товаров, корзину покупок, личный кабинет и поиск по товарам.

### Особенности проекта

- Полностью статический сайт (HTML/CSS/JS)
- Работает на GitHub Pages
- Корзина товаров с сохранением в localStorage
- Поиск по товарам
- Личный кабинет (регистрация/вход)
- Фильтрация товаров по категориям
- Адаптивный дизайн

---

## 📁 Структура сайта

- index.html - Главная страница
- catalog.html - Каталог товаров
- product.html - Карточка товара
- cart.html - Корзина
- search.html - Поиск
- account.html - Личный кабинет
- delivery.html - Доставка
- payment.html - Оплата
- returns.html - Возврат и гарантия
- 404.html - Страница ошибки

---

## 🛍️ Функционал

**Главная страница** - Hero-блок, новинки, популярное
**Каталог** - Сетка товаров, фильтр по категориям
**Карточка товара** - Описание, характеристики, отзывы
**Корзина** - Добавление, удаление, изменение количества
**Поиск** - Поиск по названиям и описаниям
**Личный кабинет** - Регистрация, вход, профиль, заказы
