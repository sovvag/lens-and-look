## 📖 О проекте

**Lens & Look** — это полностью адаптивный интернет-магазин оптики, разработанный в рамках учебного проекта. Сайт предоставляет удобный каталог товаров, корзину покупок, личный кабинет и поиск по товарам.

### 🎯 Особенности проекта

- ✅ Полностью статический сайт (HTML/CSS/JS)
- ✅ Работает на GitHub Pages
- ✅ Корзина товаров с сохранением в `localStorage`
- ✅ Поиск по товарам в реальном времени
- ✅ Личный кабинет (регистрация/вход)
- ✅ Фильтрация товаров по категориям
- ✅ Адаптивный дизайн для всех устройств
- ✅ Современный UI с плавными анимациями

## 🛍️ Функционал сайта

### 🔹 Главная страница
- Hero-блок с призывом к действию
- Блоки "Новые поступления" и "Популярное"
- Быстрый переход в каталог

### 🔹 Каталог
- Сетка товаров с картинками и ценами
- Фильтрация по категориям (линзы, очки, солнцезащитные, ночные)
- Клик по карточке → переход на страницу товара

### 🔹 Карточка товара
- Подробное описание и характеристики
- Выбор количества
- Кнопка "В корзину"
- Блок с отзывами

### 🔹 Корзина
- Список добавленных товаров
- Изменение количества
- Удаление товаров
- Автоматический подсчёт суммы
- Сохранение корзины в браузере

### 🔹 Поиск
- Поиск по названиям и описаниям товаров
- Мгновенные результаты

### 🔹 Личный кабинет
- Регистрация и вход
- Редактирование профиля
- История заказов
- Избранное

### 🔹 Информационные страницы
- Доставка
- Оплата
- Возврат и гарантия
